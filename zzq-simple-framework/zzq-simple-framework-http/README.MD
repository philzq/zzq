## 主流HttpClient框架比较
- 主要研究过JDK及OkHttp，以下对HttpComponents的评价可能不够客观。
- google反对HttpComponents因设计复杂，而认可OkHttp且默认继承到android。
- 大家都认可google，所以选OkHttp不会错。
- 虽然现在[排行第二](https://mvnrepository.com/open-source/http-clients)，但他比第一出来晚多了，足见大势所趋。
